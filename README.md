# R5V Visual Script Editor

This is a standalone visual script editor for Apex Legends R5V.

## Running the application

To run the application, you need to have Node.js and npm installed.

1.  Install the dependencies:
    ```bash
    npm install
    ```
2.  Run the development server:
    ```bash
    npm run dev
    ```

This will start the application in a new Electron window.
